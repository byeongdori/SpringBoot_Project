# SpringBoot_Project
- ### SpringBoot Practice Project  
  간단한 웹 상품 주문 어플리케이션 개발을 통해 스프링부트, JPA 기술 활용하고 응용해보기

Start 2022.01.12  
End 2022.01.24

## TOPIC
- 프로젝트 세팅
  - 스프링 부트, JPA, 타임리프, lombok, H2 데이터베이스
  
- 도메인 분석 설계  
  - 요구사항 분석
  - 도메인 모델 & 테이블 설계
  - JPA 활용한 엔티티 클래스 개발
  - 엔티티 클래스와 테이블 매핑

- 도메인 개발  
  - 비즈니스 로직
  - Repository, Service 개발
  - 테스트 케이스를 통한 검증

- 웹 계층 개발
  - Controller, 타임리프 뷰 템플릿
  - 변경 감지와 병합
